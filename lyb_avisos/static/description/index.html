Configuración

Configuración básica:

  1. Avisos > Configuración > Canales de venta
  2. Seleccione el canal de venta 'Aviso' y pulse Activo/Archivar

De forma opcional podemos agregar los usuarios que vayan a usar los avisos al canal de venta 'Aviso'

  1. Avisos > Configuración > Canales de venta
  2. Seleccione el canal de venta 'Aviso' y pulse Editar
  3. Miembros > Añadir
  4. Seleccionamos los usuarios elegidos para el uso y pulse Seleccionar A continuación pulse Guardar

Se pueden usar las etapas predefinidas o definir unas propias. Para definir sus propias etapas:

  1. Avisos > Configuración > Etapas. Pulse Crear
  2. Introduzca un nombre para la Etapa.

Uso

Vaya al menu Avisos. Podremos crear uno, para ello pulse Crear, ó seleccione el aviso ya creado.

Técnico

Para poder diferenciar entre una oportunidad y un aviso, se agrega un nuevo campo Selection al modelo el cual contiene dos valores uno para los avisos y otro para las oportunidades. Este campo además se usa para seleccionar que campos deben verse en los avisos y cuales en las oportunidades.

Nota: si necesitaramos diferenciar algún documento más que fuese similar al aviso, aumentariamos los valores de dicho campo y extenderiamos la funcionalidad.
